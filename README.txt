

This is the book Real-Time Communications Quick Start Guide published
at http://rtcquickstart.org

The book is authored using DocBook 5 markup.

The make-book.sh script builds the HTML and PDF versions of the book.

Before running it for the first time, make sure you install the
required utilities.  On a Debian/Ubuntu system, they can be
installed with:

$ sudo apt-get install xsltproc fop docbook-xsl
